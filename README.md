# 西脇 徹 （Toru Nishiwaki）

訪問介護の現場経験をもとに、介護業務の中にある課題をデータ分析・AI活用・業務改善の観点から検証しています。
現在は訪問介護職からITコンサルタント職へのキャリアチェンジを目指し、現場で感じた課題をポートフォリオという形で可視化しています。
